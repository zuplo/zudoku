import { Input } from "zudoku/ui/Input";

An input component for forms and user input.

## Import

```tsx
import { Input } from "zudoku/ui/Input";
```

## Basic Usage

<Input placeholder="Enter your email" />

```tsx
<Input placeholder="Enter your email" />
```

## Input Types

### Text Input

<Input type="text" placeholder="Text input" />

```tsx
<Input type="text" placeholder="Text input" />
```

### Email Input

<Input type="email" placeholder="Email input" />

```tsx
<Input type="email" placeholder="Email input" />
```

### Password Input

## Custom Styling

The Input component accepts all standard input HTML attributes and can be
customized with additional `className`:

```tsx
<Input className="w-full max-w-xs" placeholder="Custom styled input" />
```
