# `shadcn/ui` components

We use a couple components from [shadcn/ui](https://ui.shadcn.com). To make them re-usable for and use them in your project you can import them from `zudoku/ui/*`.

For example:

```tsx
import { Button } from "zudoku/ui/Button";
```

Following components are available:

<ul className="grid grid-cols-2 sm:grid-cols-3">
  {components.map((comp) => {
    const link = comp.replace(/([a-z])([A-Z])/g, "$1-$2").toLowerCase();

    return (
      <li key={comp}>
        <a
          target="_blank"
          href={`https://ui.shadcn.com/docs/components/${link}`}
        >
          {comp}
        </a>
      </li>
    );

})}

</ul>
