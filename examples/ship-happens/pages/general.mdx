## Services & Shipping

### Global Shipping

With the Ship Happens API, you can:

- Create and manage shipments
- Generate shipping labels and stamps
- Track shipments in real-time
- Receive webhook notifications for shipment status updates
- Access multiple shipping carriers through a single integration

### Interplanetary Shipping

All API requests require authentication using an API key, which should be included in the `X-API-Key` header. To obtain an API key, please contact our [support team](mailto:api@sh.example.com).
