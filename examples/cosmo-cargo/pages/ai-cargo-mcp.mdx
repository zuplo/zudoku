# AI Cargo <span className="text-primary">MCP</span> Server

Cosmo Cargo's <strong>Model Context Protocol</strong> server lets your favorite AI assistants
hyperlink directly into our quantum cargo grid — request route optimizations, query manifests, and
schedule warp jumps without leaving the chat window.

The configuration card below is rendered with the standalone `<MCPEndpoint />` component, so you can
drop it into any MDX page that talks about an MCP server — no OpenAPI schema required.
