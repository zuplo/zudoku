## Algolia DocSearch Example

This example demonstrates the Algolia DocSearch integration with Zudoku.

Press **Cmd+K** (or **Ctrl+K** on Windows/Linux) to open the search modal.

### Configuration

The search is configured in `zudoku.config.ts` with the `type: "algolia"` search option. This
example uses Algolia's public demo credentials for testing purposes.
